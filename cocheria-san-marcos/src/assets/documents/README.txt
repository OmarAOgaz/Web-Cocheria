INSTRUCCIONES PARA AGREGAR EL DOCUMENTO DE ACOMPAÑAMIENTO
===========================================================

Para que la sección de "Acompañamiento" funcione correctamente, 
debe colocar el archivo PDF en esta carpeta con el siguiente nombre:

    guia-acompanamiento-duelo.pdf

Si desea usar un nombre diferente, debe actualizar el enlace en el archivo:
    src/index.html

Busque la línea:
    <a href="assets/documents/guia-acompanamiento-duelo.pdf" ...

Y cambie el nombre del archivo por el que desee usar.
