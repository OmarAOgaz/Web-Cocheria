# Cochería San Marcos

Welcome to the Cochería San Marcos project! This website serves as a comprehensive resource for families seeking funeral services, providing clear information and support during difficult times.

## Project Structure

- **src/**: Contains the main source files for the website.
  - **index.html**: The entry point of the website, featuring the main slider and navigation.
  - **css/**: Contains stylesheets for the website.
    - **styles.css**: Main styles for layout and appearance.
    - **responsive.css**: Styles for responsive design.
  - **js/**: Contains JavaScript files for interactivity.
    - **main.js**: Main functionality and event listeners.
    - **slider.js**: Manages the main slider functionality.
  - **assets/**: Contains assets such as fonts.
    - **fonts/**: Directory for font files.

- **pages/**: Contains individual HTML pages for different sections of the website.
  - **services.html**: Information about the services offered.
  - **about.html**: Details about the company and its mission.
  - **viewing-rooms.html**: Information on available viewing rooms.
  - **faq.html**: Frequently asked questions and answers.
  - **support.html**: Information on support services.
  - **contact.html**: Contact information and inquiry form.

## Installation

To set up the project locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd cocheria-san-marcos
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Open `src/index.html` in your web browser to view the website.

## Usage

This website is designed to provide a user-friendly experience, with a clear navigation structure that includes:

- A main slider showcasing key services and information.
- Sections for services, about us, viewing rooms, FAQs, support, and contact information.
